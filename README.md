# 📄 Alfredo Mungari - Professional Resume (LaTeX)

Welcome to the repository containing the LaTeX source code for my professional resume. 

As a **Data Scientist, ML Expert, and Software Engineer**, I built this CV to strike the perfect balance between a premium, modern design and strict **ATS (Applicant Tracking System)** compatibility.

## ✨ Key Features

* 🤖 **100% ATS-Friendly:** Built with a linear, single-column structure. It avoids complex tables or floating elements that confuse parsing algorithms.
* 🔤 **Unicode Mapping:** Uses the `cmap` package to ensure that PDF text extraction works flawlessly, meaning every keyword is correctly read by recruitment software.
* 🎨 **Premium Typography:** Leverages `microtype` for perfect text justification and `setspace` for an airy, elegant line height.
* 🔗 **Fully Interactive:** Clickable hyperlinks for contact info, GitHub repositories, LinkedIn, and research publications.
* 🖌️ **Custom Color Palette:** Uses a custom RGB palette (Deep Navy, Slate Gray, Tech Steel Blue) to establish a clear visual hierarchy.

## 🚀 How to Compile

You can easily compile this resume using your preferred LaTeX environment.

### Option 1: Overleaf (Recommended)
1. Create a new blank project on [Overleaf](https://www.overleaf.com/).
2. Copy the contents of `resume.tex` into the main file.
3. Set the compiler to **pdfLaTeX**.
4. Click "Recompile".

### Option 2: Local Environment
If you have a local TeX distribution (like TeX Live or MiKTeX) installed, open your terminal and run:
```bash
pdflatex resume.tex
```

## 📦 Main LaTeX Packages Used
* `cmap`, `inputenc`, `fontenc`, `babel` (Text extraction and encoding)
* `microtype`, `setspace` (Advanced typography)
* `xcolor` (Custom color definitions)
* `geometry`, `titlesec` (Layout and section formatting)
* `enumitem` (Custom bullet points)
* `hyperref` (Clickable links)
